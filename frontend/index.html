Alur Bisnis RFID

  1. Admin mengetukkan kartu RFID fisik ke reader ESP32
  2. ESP32 membaca UID kartu dan mengirimkan via HTTP ke backend
  3. Admin mengisi nama produk, harga, dan stok awal
  4. UID tersebut permanen dipetakan ke produk tersebut
  5. Untuk update stok, cukup ubah kuantitas di menu Inventori

Aturan Penting

Satu UID RFID hanya dapat didaftarkan untuk satu produk. Ini dirancang agar admin cukup mengetuk satu kartu untuk satu jenis produk, bukan satu kartu per item.

POST http://localhost:3000/products

Mendaftarkan produk baru dengan UID RFID

{
  "uid": "ABCD1234",
  "name": "Batik Tulis Pekalongan",
  "stock": 50,

Memperbarui stok produk (tambah / kurangi / set)

{
  "uid": "ABCD1234",
  "quantity": 25,
  "operation": "add"
